Hot Beans Web - Job Specifications

Frontend Developer

Responsibilities:

  • Develop and maintain the front end of web applications.
  • Ensure the technical feasibility of UI/UX designs.
  • Optimize applications for maximum speed and scalability.
Requirements:
  • Proficient in HTML, CSS, JavaScript, and modern frameworks like React or Angular.
  • Strong understanding of web design principles.
  • Experience with responsive and adaptive design.

Backend Developer

Responsibilities:

  • Build and maintain server-side applications and APIs.
  • Ensure the reliability and performance of web applications.
  • Collaborate with frontend developers to integrate user-facing elements.
Requirements:
  • Proficient in server-side languages such as Node.js, Python, or Ruby.
  • Experience with database management and design (SQL, NoSQL).
  • Strong understanding of RESTful API design principles.

Full Stack Developer

Responsibilities:

  • Develop both client and server software.
  • Design and implement full-stack solutions.
  • Collaborate with cross-functional teams to define and design new features.
Requirements:
  • Proficient in both frontend and backend technologies.
  • Experience with databases, RESTful services, and API design.
  • Strong problem-solving skills and attention to detail.

UI/UX Designer

Responsibilities:

  • Design and create user interfaces and user experiences for web applications.
  • Conduct user research and create wireframes, prototypes, and mockups.
  • Collaborate with developers to ensure the implementation of designs.
Requirements:
  • Proficient in design software like Sketch, Figma, or Adobe XD.
  • Strong portfolio of design projects.
  • Good understanding of user-centered design principles.

Project Manager

Responsibilities:

  • Plan, execute, and oversee web development projects.
  • Coordinate between teams and stakeholders.
  • Ensure projects are delivered on time and within budget.
Requirements:
  • Experience in project management and web development.
  • Strong leadership and organizational skills.
  • Proficient in project management tools like JIRA or Trello.
